# صالة العرض | Arabic Showroom Catalog - مع لوحة تحكم

موقع كتالوج عربي فاخر للعرض فقط مع **لوحة تحكم كاملة** لإدارة المنتجات.

---

## 🚀 التشغيل السريع

```bash
# 1. تثبيت التبعيات
npm install

# 2. إعداد البيئة
# انسخ ملف .env.local.example إلى .env.local وعدل القيم:
cp .env.local.example .env.local

# 3. تشغيل خادم التطوير
npm run dev

# 4. فتح الموقع
# http://localhost:3000

# 5. فتح لوحة التحكم
# http://localhost:3000/admin
# كلمة المرور الافتراضية: admin123
```

---

## 🔐 إعداد كلمة المرور

افتح ملف `.env.local` وعدل القيم:

```env
ADMIN_PASSWORD=your-secure-password-here
JWT_SECRET=any-random-long-string-here-change-it
```

| المتغير | الوصف |
|---------|-------|
| `ADMIN_PASSWORD` | كلمة المرور لتسجيل الدخول للوحة التحكم |
| `JWT_SECRET` | مفتاح سرّي لتوقيع الكوكيز (أي نص عشوائي طويل) |

## 📊 لوحة التحكم (/admin)

### الصفحات
| الصفحة | الوصف |
|--------|-------|
| `/admin/login` | صفحة تسجيل الدخول |
| `/admin` | لوحة التحكم الرئيسية |

### التبويبات

#### 1️⃣ المنتجات
- عرض جميع المنتجات في جدول
- **بحث فوري** بالاسم أو الفئة
- **تعديل** أي منتج (اضغطي على أيقونة القلم)
- **حذف** منتج (أيقونة السلة)
- **إضافة تخفيض** (أيقونة الوسم) — تفتح نافذة تدخلين فيها السعر القديم
- **إزالة تخفيض** (أيقونة إلغاء الوسم)
- **تبديل وسم "جديد"** (أيقونة النجمة)
- **عرض المنتج** في الموقع (أيقونة السهم)

#### 2️⃣ إضافة منتج
- نموذج كامل لإضافة منتج جديد
- الحقول: الاسم، الفئة، السعر، السعر القديم، الوصف، الألوان، المقاسات، الصور، الوسوم
- الألوان والمقاسات والصور والوسوم تُفصل بفواصل `,`

#### 3️⃣ الإحصائيات
- إجمالي المنتجات
- عدد المنتجات بتخفيض
- عدد المنتجات الجديدة
- عدد التصنيفات
- رسم بياني لتوزيع المنتجات حسب الفئة

## 💰 كيفية إضافة تخفيض

### الطريقة 1: من لوحة التحكم (الأسهل)
1. ادخلي `/admin` وسجلي الدخول
2. اذهبي لتبويب **المنتجات**
3. اضغطي على أيقونة الوسم 🏷️ بجانب المنتج
4. أدخلي السعر القديم (الأصلي قبل التخفيض)
5. اضغطي "تأكيد التخفيض"

### الطريقة 2: يدوياً في products.json
```json
{
  "id": "1",
  "name": "فستان سهرة",
  "price": 450,        // ← السعر الجديد (بعد التخفيض)
  "oldPrice": 599,     // ← السعر القديم (قبل التخفيض)
  "tags": ["تخفيض"]    // ← أضفي وسم التخفيض
}
```

الموقع يحسب نسبة الخصم تلقائياً: `((599 - 450) / 599) × 100 = 25%`

## ➕ كيفية إضافة منتج جديد

### الطريقة 1: من لوحة التحكم (الأسهل)
1. `/admin` → تبويب **إضافة منتج**
2. املئي النموذج
3. للألوان: `أسود, أبيض, بيج`
4. للمقاسات: `S, M, L, XL`
5. للصور: `https://images.unsplash.com/..., https://...`
6. للوسوم: `جديد, تخفيض` أو اتركيها فارغة

### الطريقة 2: يدوياً في products.json
أضفي كائن JSON جديد في نهاية المصفوفة (قبل `]`):

```json
,{
  "id": "17",
  "name": "اسم المنتج بالعربي",
  "category": "ملابس نسائية",
  "price": 250,
  "oldPrice": 350,
  "description": "وصف المنتج هنا...",
  "colors": ["أسود", "أبيض"],
  "sizes": ["S", "M", "L"],
  "images": ["https://images.unsplash.com/photo-..."],
  "tags": ["جديد"]
}
```

## 🗑️ كيفية حذف منتج

### الطريقة 1: من لوحة التحكم
1. `/admin` → تبويب **المنتجات**
2. اضغطي على أيقونة السلة 🗑️ بجانب المنتج
3. أكدي الحذف

### الطريقة 2: يدوياً
احذفي كائن المنتج من ملف `data/products.json`

---

## 🔄 كيف يعمل تحديث products.json

لوحة التحكم تستخدم **Server Actions** (وظائف الخادم):

```
النقر في المتصفح
    ↓
Server Action (تشغل على الخادم)
    ↓
تقرأ products.json ← fs.readFileSync()
    ↓
تعدل البيانات
    ↓
تكتب products.json → fs.writeFileSync()
    ↓
revalidatePath('/') ← تحديث الصفحات تلقائياً
```

**لا تحتاجين لإعادة تشغيل الخادم!** التغييرات تظهر فوراً.

## 🛡️ الأمان

| الميزة | الوصف |
|--------|-------|
| Middleware | يحمي `/admin/*` ويمنع الوصول بدون تسجيل دخول |
| JWT Cookies | كوكيز `httpOnly` + `secure` + `sameSite` |
| Expiration | الجلسة تنتهي بعد 8 ساعات |
| Password-only | لا يوجد قاعدة بيانات — فقط كلمة مرور في `.env` |
